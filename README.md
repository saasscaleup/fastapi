## Installation

Follow next steps in order to have FastAPI service working for you!

### Step 1 - Git clone

```
git clone https://github.com/saasscaleup/fastapi.git
```

```
cd fastapi
```

### Step 2 - Install required packages.


Install [Fastapi](https://fastapi.tiangolo.com/) packages.
```
pip install fastapi uvicron
```

## Run FastAPI locally

```
uvicorn main:app --reload
```
